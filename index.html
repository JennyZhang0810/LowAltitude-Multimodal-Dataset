UAIV

A Large-Scale Low-Altitude Multimodal Dataset for Urban Intelligence
📄 Paper 💻 Code 📦 Dataset

🔥 Highlights

🚀 17B+ annotations
🌐 Cross-modal (RGB / IR)
🌧 Multi-weather pairing
🕒 Multi-temporal alignment
✈ UAV flexible acquisition
🧠 Physics-aware modeling

🔥 Overview

UAIV is designed for real-world urban intelligence, enabling models to learn robust representations across weather, time, and viewpoints, with strong emphasis on physical consistency.

🧠 Dataset Pipeline

UAV acquisition → multi-modal alignment → annotation → multi-task learning

🧩 Task Definitions

1. Scene Understanding

2. Spatio-Temporal Learning

3. Physics-Aware Restoration
