UAIV: A Large-Scale Low-Altitude Multimodal Dataset for Urban Intelligence

Multi-scenario • Multi-modal • Multi-condition • Real-world UAV Intelligence

From perception → reasoning → physics-aware understanding under real-world UAV conditions

--- ## 🔥 Overview **UAIV** is a large-scale low-altitude multimodal dataset designed for **urban intelligence and fine-grained governance analysis**, with a strong focus on: - **Scene understanding** - **Spatio-temporal reasoning** - **Physics-aware image restoration** Unlike conventional UAV datasets, UAIV is built around a key principle: > **Learning consistent representations across modality, condition, and time** This is achieved through: - **Cross-modal alignment** (RGB / IR / metadata) - **Cross-view consistency** - **Cross-condition robustness** (weather / illumination / viewpoint) The dataset is constructed via a **self-operated UAV system**, enabling **long-term, large-scale, real-world data collection** with strong diversity in: - Scene types (urban / rural / industrial / natural) - Illumination (day / night) - Weather (rain / snow / fog / haze) - Flight altitude and trajectories --- ## 🚀 What Makes UAIV Different > Not just a dataset — but a **unified paradigm for real-world multimodal intelligence** ### 1️⃣ Beyond Multi-Modal → **Cross-Condition Alignment** Unlike existing datasets that treat modalities independently, UAIV provides: - **Aligned RGB–IR–metadata signals** - **Consistent sampling across weather, time, and viewpoints** This enables learning **invariant representations under real-world distribution shifts**, rather than overfitting to static conditions. --- ### 2️⃣ Beyond Perception → **Perception + Reasoning + Physics** UAIV is one of the few datasets that jointly supports: - Visual perception (segmentation, detection, OCR) - Spatio-temporal reasoning (change detection, Re-ID) - **Physics-aware modeling** (weather degradation & restoration) This bridges three traditionally separated research areas into a **single unified benchmark**. --- ### 3️⃣ Beyond Benchmark → **Real-World Deployment** UAIV is not a purely academic dataset. It has been: - **Deployed in real urban governance systems** - Used to build **21 domain-specific intelligent agents** - Supporting **large-scale regional monitoring platforms** This makes UAIV a rare dataset that connects: > **Data → Model → System → Real-world impact** --- ## ✨ Highlights - **17B+ high-quality annotations** - **Cross-modal & cross-view alignment** (RGB / IR / metadata) - **Multi-weather paired data** for restoration (rain / snow / fog) - **Multi-temporal alignment** for change detection and Re-ID *(coming soon)* - **Flexible UAV acquisition**: multi-altitude, multi-route, repeated sampling - **Unified multi-task data paradigm** across perception, reasoning, and restoration - **Object-level and scene-level consistency modeling** --- ## 📊 Dataset Overview - **Scene diversity**: urban / rural / industrial / natural - **Weather conditions**: sunny / rainy / foggy / snowy - **Temporal coverage**: day / night - **Metadata support**: GPS, timestamp, altitude, camera parameters --- ## 🧩 Task Definitions --- ### 1. Scene Understanding **Tasks** - Scene classification - Semantic segmentation - Instance segmentation - Object counting - OCR - Environment understanding **Description** UAIV goes beyond conventional benchmarks by emphasizing **context-rich and governance-oriented perception**. Each sample corresponds to real-world scenarios such as: - Illegal construction - Open burning - Environmental pollution - Infrastructure monitoring With **multi-modal alignment and environmental diversity**, models must jointly reason about: - Object presence - Scene semantics - Environmental status --- **Examples**

--- ### 2. Spatio-Temporal Learning **Tasks** - Change detection - Cross-time Re-ID **Description** UAIV enables **long-term dynamic scene understanding** with aligned multi-temporal data. Models must distinguish: - **True semantic changes** - vs. **appearance variations** caused by weather or lighting It also introduces **cross-time Re-ID under extreme viewpoint and scale variations**, pushing identity learning beyond static settings. --- **Examples**

--- ### 3. Physics-Aware Image Restoration **Tasks** - Rain / snow / fog removal - Cross-weather image translation - Weather-aware degradation modeling **Description** UAIV provides **real-world weather-degraded data** with preserved physical consistency: - Atmospheric scattering - Illumination variation - Sensor response differences This enables models to recover: > **Intrinsic scene properties instead of superficial appearance** Cross-modal signals (RGB–IR) further support **physically grounded restoration**. --- **Examples**
