🔥 Overview

UAIV is a large-scale low-altitude multimodal dataset designed for urban intelligence and fine-grained governance analysis.

It focuses on scene understanding, spatio-temporal reasoning, and physics-aware image restoration, enabling models to learn physically grounded representations under real-world complexity.

Unlike conventional UAV datasets, UAIV emphasizes:

  • Cross-modal alignment (RGB / IR / metadata)
  • Cross-view consistency
  • Cross-condition robustness (weather / illumination / viewpoint)

🧩 Task Definitions

1. Scene Understanding

  • Scene classification
  • Semantic segmentation
  • Instance segmentation
  • Object counting
  • OCR
  • Environment understanding

UAIV emphasizes context-rich and governance-oriented perception, enabling models to jointly reason about objects, scenes, and environmental states.

2. Spatio-Temporal Learning

  • Change detection
  • Cross-time Re-ID

Supports aligned multi-temporal data across time, weather, and viewpoints, enabling models to distinguish true semantic changes from appearance variations.

3. Physics-Aware Image Restoration

Provides real-world weather-degraded data (rain, snow, fog, haze) with preserved physical consistency.

Supports multi-modal restoration (RGB–IR) and physically grounded enhancement.

🚀 Applications & Impact

UAIV has been deployed in real-world urban governance systems in Xuzhou, covering large-scale regional monitoring.

  • Illegal construction detection
  • Environmental monitoring
  • Infrastructure assessment

It significantly improves event detection accuracy and reduces manual inspection cost.
